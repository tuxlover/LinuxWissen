locale

Created Mittwoch 14 September 2011

  • mit dem Befehl locale lassen sich Lokalisierungsspezifische Informationen verwalten

locale [<LC_VAR>] → verwalten von Lokalisierungsspezifischen Informationen

→ wird eine spezille LC-Variable als Parameter übergen, so werden Informationen über diese LC_Variablen ausgegeben
→ wird keine spezielle LC_Variable als Parameter übergeben, so werden Informationen über alle derzeit gesetzten LC-Variablen ausgegeben

-a → alle Verfügbaren Werte für die Lokalisierungsvariablen ausgeben
-k <LC_VAR> → zeigt die tatsächlichen Einstellungen für < LC_VAR> an

Hinweis: Beim Anmelden eines neuen Nutzers muss stets die Umgebung des Nutzers geladen sein.
