Splitten

Created Dienstag 19 April 2011

split <Sehr_große_Datei> → eine <Sehr_große_Datei> in mehrere kleinere zerteilen
-a → Suffix festlegen
-b Nb|K|MB|M|G|GB|T|P → Größe in Bytes pro Teil
-d → nummerischer Suffix anstelle alphabetische Suffix

Beispiel:
split -a name_ -b 2GB <Große_Datei>
