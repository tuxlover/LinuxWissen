  • Linux für Menschen mit Behinderungen zugänglich machen

klebenede Tasten (stickey keys)
→ Wird die <Strg>- Taste einmal gedrückt, wird sie als gedrückt gehalten interpreteirt
→ Wird die <Strg>-Taste ein weiteres mal gedrückt wird sie als gelöst interpretiert

langsame Tasten (slow keys)
→ ignorieren von Tastendrücken innehalb eines zeitlich festgelegten Intervalls

zurückschnellende Tasten (bounce keys)
→ überzählige Tastendrücke derselben Taste werden ignoriert

Wiederholungstasten (repeat keys)
→ sollen festhaltene Tasten wiederholt oder nur einmal ausgegeben werden

Maustasten (mouse keys)
→ gestattet die Steuerung eines Mauszeigers über den Nummernblock

visuelle Keyboards
→ Bildschirmtastatur die mit er Maus gesteuert werden kann
