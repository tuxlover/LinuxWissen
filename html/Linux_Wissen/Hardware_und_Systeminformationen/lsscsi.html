lscsi [<Host>:<Channel>:<ID>:<LUN>] → Informationen über SCSI Geräte anzeigen

→ Jedes Gerät kann entsprechend nach H:C:I:L gefiltert werden
-c|--classic → cat /proc/scsi Linux Wissen:FHS:proc
-H → anzeigen der momentanen angeschlossenen Geräte
-l → zusätzliche ausführliche Informationen anzeigen
