Escape Sequenzen

Created Donnerstag 28 April 2011

\b
→ backspace (^H)
\f
→ formfeed (^L)
\n
→ newline (DOS, CR/LF; Unix, LF)
\r
→ carriage return
\t
→ tab (^I)
\ddd
→ oktaler Wert `ddd' wobei `ddd' eine 1-3 stellige Zahl von 0 bis 7 ist
\c
→ Jedes andere Zeichen würd durch ein Literal gesehen , \" für " und \\ für \
