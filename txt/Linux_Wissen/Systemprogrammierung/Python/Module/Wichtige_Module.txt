====== Wichtige Module ======
Created Mittwoch 21 August 2013

**math**
→ Modul für mathematische Berechnungen
→ enthält zum Beispiel Pi 
→ enthält mathematische Funkitonen wie //Sin, Cos, Exp//

**random**
→ Zufalls generator
→ //randint, randrange, random//

**re**
→ Module für Pattern Matching und Reguläre expressions
