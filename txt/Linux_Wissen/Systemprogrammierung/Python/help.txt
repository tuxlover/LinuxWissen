====== help ======
Created Mittwoch 21 August 2013

* Teil von //Pydoc//
* Die Hilfsfunktion in Python ähnlich der manpages in Linux
* Mit der Hilfsfunktion kann man scih eine Übersicht bzw Hilfe zu einer bestimmten:
	* Mehtode
	* Objekt
	* Paradigma

**help()**
→ die Übersicht aufrufen

**help(<Name>)**
→ gibt Hilfe zu Name
