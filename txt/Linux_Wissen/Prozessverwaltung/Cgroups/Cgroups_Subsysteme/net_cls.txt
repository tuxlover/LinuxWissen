====== net cls ======
Created Mittwoch 01 Februar 2012

* vergibt  //ClassID// Markierungen für Netzwerkpakete 
* der Linux Traffic Controller kann Pakte die von diesem Paket aus gesendet weren zuordnen
* Der Traffic Controller kann so konfiguriert werden, dass verschiedene Pakete verschiedenen Cgroups zugeornet werden kann

**net_cls.classid**
→ Wertebereich: einzelener Wert Hexadezimal
→ verweist auf ein //Traffic Controle Handle//
**(missing Link tc)**
