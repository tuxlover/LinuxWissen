====== Prompt ======
Created Mittwoch 09 November 2011

* bietet mehrere Prompts die den Status eines Statements angeben

''mysql> ''→ wartet auf Eingabe

''-> ''→ Wartet auf die nächste Zeile im Statement

'''> ''→ Wartet auf das Ende eines eine '' Quotes

''"> ''→ Wartet auf das Ende eine " " Quotes

''`> ''→ Warten auf das Ende des Backticks

''/*>'' → Warten auf das Ende eines Kommentars
