====== Mathematische Funktionen ======
Created Freitag 10 Juni 2011

* mathematische und Aggregatfunktionen können beliebig miteinander kombiniert werden

**SELECT <spalte> * <Wert> FROM <tabelle> **
→ multiplizieren Spalte mit <Wert>

**SELECT <spalte> + <Wert> FROM <tabelle>**
**→ **addieren von <Wert> zu 

**SELECT <spalte> - <Wert> FROM <tabele> **
→ subtrahiere <Wert> Spalte

**SELECT <spalte> / <Wert> FROM <tabelle>**
→ dividiere <spalte> durch <Wert> 


== ABS() ==

**SELECT ABS( <Ausdruck> )**
	**FROM <tabelle> ...**
→ gibt den Absoluten Wert | die Metrik  eines <Ausdruck> von <tabelle> zurück

== EXTRACT() ==

**SELECT EXTRACT( {DAY | MONTH | YEAR} FROM <spalte> ) **
	**FROM <tabelle>**
→ gibt Tag, Monat oder Jahr als Zahl von einer <spalte> vom Datentyp <Date> aus

== MOD() ==

**SELECT MOD(<spalte>,N) **
	**FROM <tabelle>**
→ gibt den Modulowert bei der Devision der <Spalte> durch N zurück

== POSITION() ==

**SELECT POSITION ( '<zeichen>' IN <spalte)**
	**FROM <tabelle>**
→ zeigt die Postion eines bestimmten Zeichens in einer bestimmen <spalte> von <tabelle>
