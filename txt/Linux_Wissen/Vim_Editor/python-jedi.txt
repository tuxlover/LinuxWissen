====== python-jedi ======
Created Donnerstag 02 Oktober 2014

* ist ein addin für die programmiersprache python ( [[Linux Wissen:Systemprogrammierung:Python]] )
* erlaubt die einfache Benutzung von Autocompletion etc.

=== How to use: ===
**→ nach** **der installation über den paketmanager muss das plugin in vim integriert werden**
''vim-addons install python-jedi''
**:help**
**<shift>-k** -> auf die installierte Dokumentation zurückgreifne
**<strg>-leer** -> autocompletion
